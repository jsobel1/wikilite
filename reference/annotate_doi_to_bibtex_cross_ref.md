# Retrieve BibTeX entries for a list of DOIs via CrossRef

Retrieve BibTeX entries for a list of DOIs via CrossRef

## Usage

``` r
annotate_doi_to_bibtex_cross_ref(doi_list)
```

## Arguments

- doi_list:

  Character vector of DOIs.

## Value

A list of BibTeX character strings, one per DOI.

## Examples

``` r
if (FALSE) { # \dontrun{
annotate_doi_to_bibtex_cross_ref("10.1038/nature12373")
} # }
```
